## First time install

Steps to run the project locally

#### Install Jekyll

Follow directions at [jekyllrb.com](https://jekyllrb.com)


```

#### Install Jekyll Gem

```
$ gem install jekyll
```

#### Install Bundler Gem

```
$ gem install bundler
```

#### Run Bundle Install

```
$ bundle install
```

#### Running locally
```
$ bundle exec jekyll serve --watch --trace
```
